<b> 
To install the dependencies used for the entire Frontend part, please run : : </b>

```
npm create vite@latest
npm install
npm run dev

npm init 
npm install -D tailwindcss postcss autoprefixer
npx init tailwindcss -p

npm install react-icons ethers

```
